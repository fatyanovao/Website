📌 Задача 12 (минимальная чётная цифра)


n = int(input())
min_digit = 10
while n > 0:
    digit = n % 10
    if digit % 2 == 0 and digit < min_digit:
        min_digit = digit
    n //= 10
if min_digit == 10:
    print("NO")
else:
    print(min_digit)

✅ При вводе 231 → выведет "NO" (нет чётных цифр).

📌 Задача 4 (скорость передачи данных)

Документ 16 Мбайт = 16 × 2²³ бит. Скорость 2²¹ бит/с.

  • Способ А: сжатие 25% = 4 Мбайт. Время = 12 + (4×2²³ / 2²¹) + 3 = 12 + 16 + 3 = 31 сек
  • Способ Б: без сжатия = 16×2²³ / 2²¹ = 64 сек
  • Ответ: А33 (быстрее на 33 секунды)
